  ★  レッスン: 基本移動コマンド  ★

  ノーマルモード (Esc を押してノーマルモードに入る) では、
  以下のキーでカーソルを動かせる。矢印キーは使用禁止！

  ┌─────────────────────────────────────────┐
  │  h   ←   左に1文字移動                  │
  │  l   →   右に1文字移動                  │
  │  j   ↓   下に1行移動                    │
  │  k   ↑   上に1行移動                    │
  ├─────────────────────────────────────────┤
  │  w       次の単語の先頭にジャンプ        │
  │  b       前の単語の先頭に戻る            │
  │  e       現在の単語の末尾にジャンプ      │
  ├─────────────────────────────────────────┤
  │  0       行の先頭にジャンプ              │
  │  $       行の末尾にジャンプ              │
  ├─────────────────────────────────────────┤
  │  gg      ファイルの最初の行にジャンプ    │
  │  G       ファイルの最後の行にジャンプ    │
  │  :42     42行目にジャンプ                │
  └─────────────────────────────────────────┘

  ヒント: コマンドの前に数字を付けられる！
       5j  = 5行下に移動
       3w  = 3単語分前に移動

  正しいルーンワードに置き換えよ。移動コマンドを使って
  そこに移動してください。

  ルーンは以下の場所にある:
    Rune 1  ([RUNE-1] のある行)  →  矢印の後の単語を探せ

    Rune 3  ([RUNE-3] のある行)  →  矢印の後の単語を探せ
