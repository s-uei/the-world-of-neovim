  ★  レッスン: 検索とコマンドライン編集  ★

  ┌────────────────────────────────────────────────────────────────┐
  │  /パターン  前方にパターンを検索                               │
  │  ?パターン  後方にパターンを検索                               │
  │  n          次のマッチにジャンプ                               │
  │  N          前のマッチにジャンプ                               │
  │  *          カーソル下の単語を検索 (前方)                      │
  │  #          カーソル下の単語を検索 (後方)                      │
  ├────────────────────────────────────────────────────────────────┤
  │  :s/古い/新しい/     現在行の最初の出現を置換                  │
  │  :s/古い/新しい/g    現在行の全ての出現を置換                  │
  │  :%s/古い/新しい/g   ファイル全体の全ての出現を置換            │
  │  :%s/古い/新しい/gc  確認しながらファイル全体を置換            │
  ├────────────────────────────────────────────────────────────────┤
  │  :w     ファイルを書き込む (保存)                              │
  │  :q     終了 (未保存の変更がある場合は失敗)                    │
  │  :wq    書き込んで終了                                         │
  │  :q!    保存せずに強制終了                                     │
  │  :e!    ファイルをリロード (変更を破棄)                        │
  ├────────────────────────────────────────────────────────────────┤
  │  :set number      行番号を表示                                 │
  │  :set nonumber    行番号を非表示                               │
  │  :set hlsearch    検索マッチをハイライト                       │
  └────────────────────────────────────────────────────────────────┘

  ★  あなたのクエスト  ★

  以下の神託のメッセージは魔法使いによって改ざんされている。
  各 [REPLACE: 古い → 新しい] が置換すべき内容を示している。

  置換コマンドを使う:
    :%s/KEYBOARD/SWORD/g   (1単語をファイル全体で置換)
  または /KEYBOARD で各単語を個別に検索して置換する。

  置換後、神託の真のメッセージが明らかになる。

  [REPLACE: NEOVIM → DESTINY ]

  ★  あなたのタスク  ★

  以下は暗号化された預言だ。暗号化された単語を
  示された通りの真の単語に置き換えよ。
  ENCODED PROPHECY (暗号化された預言) のみを編集すること:

  --- ENCODED PROPHECY (edit this section) ---

  The power of KEYBOARD flows through your fingers like lightning.
  Return to NORMAL mode as a warrior sheathes their blade.
  In INSERT mode, inscribe your will upon the digital realm.
  Let /SEARCH guide you through darkness like a lantern.
  For NEOVIM is the ancient art, reborn in modern light.

  --- END OF ENCODED PROPHECY ---

  以下の置換を上の行に実行せよ:
    :s/KEYBOARD/SWORD/g         (最初の預言行)
    :s/NORMAL/PEACE/g           (2番目の預言行)
    :s/INSERT/CREATION/g        (3番目の預言行)
    :s/\/SEARCH/WISDOM/g        (4番目の預言行)
    :s/NEOVIM/DESTINY/g         (5番目の預言行)

  またはシンプルに :%s/KEYBOARD/SWORD/g などでファイル全体に使える。
