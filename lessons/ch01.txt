  ★  THE STORY  ★

  The ancient village of Vim is burning! The evil GUI Sorcerer has
  set it ablaze with a cursed mouse-click spell. The Elder hands you
  a glowing scroll and shouts:

      "Only a true Vim Warrior can navigate this chaos!
       Learn the sacred movement arts — or perish!"

  ★  LESSON: Basic Movement Commands  ★

  In Normal mode (press Esc to enter Normal mode), you can move the
  cursor using these keys.  No arrow keys allowed in this realm!

  ┌─────────────────────────────────────────┐
  │  h   ←   move LEFT one character        │
  │  l   →   move RIGHT one character       │
  │  j   ↓   move DOWN one line             │
  │  k   ↑   move UP one line               │
  ├─────────────────────────────────────────┤
  │  w       jump to start of NEXT word     │
  │  b       jump BACK to start of word     │
  │  e       jump to END of current word    │
  ├─────────────────────────────────────────┤
  │  0       jump to start of LINE          │
  │  $       jump to END of line            │
  ├─────────────────────────────────────────┤
  │  gg      jump to FIRST line of file     │
  │  G       jump to LAST line of file      │
  │  :42     jump to line 42                │
  └─────────────────────────────────────────┘

  TIP: You can prefix commands with a number!
       5j  = move down 5 lines
       3w  = jump forward 3 words
